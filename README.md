# lbs

## Description
针对查看附近的人，提供的三种实现方式。

## Feature
 - 使用MySQL实现功能
 - 使用MongoDB实现功能
 - 使用Redis实现功能
 
## Requirement
1. Redis服务器3.2版本以上
2. MongoDB扩展使用新版：[MongoDB扩展](https://pecl.php.net/package/mongodb)
